### Установка/Запуск
1. Скачать проект.
2. Открыть папку через консоль: `cd ruby-tz-rsdigital`
3. Запустить сервер: `rails server` или `rails s`
4. Перейти по ссылке:
    [127.0.0.1:3000](http://127.0.0.1:3000/),
    [localhost:3000](http://localhost:3000/)
